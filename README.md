# Healthify_Feb25
“AI powered personal health assistant built with Streamlit and Gemini”


# demo:
https://healthifyfeb25-fevhvmlwb9fetbb8jfaepq.streamlit.app/
## Features
- Enter health details (age, weight, height, fitness rating).
- BMI calculation and fitness insights.
- Ask health-related questions and get personalized guidance.
- Uses Google Gemini for AI-powered responses.
